วิธีการติดตั้งและแก้ไขปัญหาหน้าเว็บไม่แสดง (Error 404 /src/main.tsx)
========================================================================

สาเหตุที่เปิดหน้าเว็บแล้วไม่แสดง (404 Not Found):
-------------------------------------------
คุณกำลังพยายามนำ "ไฟล์ต้นฉบับ" (Source Code) ไปรันบน PHP Server โดยตรง 
ซึ่งในระบบ React (Vite) นั้น ไฟล์เประเภท .tsx เบราว์เซอร์ไม่สามารถอ่านได้โดยตรง 
มันต้องผ่านกระบวนการ "Build" เพื่อแปลงเป็นไฟล์ HTML/JS/CSS ปกติก่อนครับ

ขั้นตอนการแก้ไข:
--------------
1. คุณต้องทำการ Build โปรเจกต์ในเครื่องคอมพิวเตอร์ของคุณก่อน:
   - เปิด Terminal หรือ Command Prompt ในโฟลเดอร์โปรเจกต์
   - พิมพ์คำสั่ง: npm install (ถ้ายังไม่เคยลง)
   - พิมพ์คำสั่ง: npm run build
   
2. หลังจากรันเสร็จ คุณจะได้โฟลเดอร์ใหม่ที่ชื่อว่า "dist"
   - โฟลเดอร์ "dist" นี้คือไฟล์ที่พร้อมใช้งานจริง (Production)
   - ให้คุณก๊อปปี้ "เนื้อหาทั้งหมดที่อยู่ในโฟลเดอร์ dist" ไปวางบน Server ของคุณครับ

3. ตรวจสอบโครงสร้างบน Server (ควรเป็นแบบนี้):
   - /index.html (ไฟล์ที่ได้มาจาก dist)
   - /assets/ (โฟลเดอร์ที่ได้มาจาก dist)
   - /api.php (ไฟล์ PHP ที่เราสร้างไว้)
   - /db.sql (สำหรับ Import ฐานข้อมูล)

การตั้งค่า api.php:
------------------
- ตรวจสอบว่าในไฟล์ api.php ตั้งค่า DB_NAME, USERNAME, PASSWORD ตรงกับโฮสติ้งแล้ว
- ใส่ Gemini API Key ในส่วนของ $api_key ในไฟล์ api.php เพื่อใช้ระบบ AI

สรุป: 
หน้าเว็บที่พยายามโหลด /src/main.tsx คือหน้าเว็บในโหมด "นักพัฒนา" (Development) 
ซึ่งจะไม่ทำงานบน PHP Server ทั่วไป คุณต้องใช้ไฟล์จากโฟลเดอร์ "dist" เท่านั้นครับ!
